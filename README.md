# Multiplica Boti 🌿
### Sistema de Treinamentos · O Boticário Niterói

Sistema para geração de avaliações de treinamento com IA, agenda de eventos e acompanhamento de resultados.

## Como rodar localmente

```bash
npm install
npm run dev
```

## Variáveis de ambiente

Crie um arquivo `.env` na raiz com:

```
VITE_ANTHROPIC_API_KEY=sua_chave_anthropic_aqui
```

## Deploy na Vercel

1. Importe o repositório na Vercel
2. Adicione a variável `VITE_ANTHROPIC_API_KEY` nas configurações
3. Deploy automático!
